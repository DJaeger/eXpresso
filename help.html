eXpresso Help

Associate filetypes with PortableApps

Description

eXpresso is a modification of the original French version of C.A.F.É. eXpresso allows a user to temportarily associate filetypes with an application that runs from a portable USB device by monitoring double-clicks on windows. All associations are recorded as relative paths in an ini file, therefore, eXpresso is completely portable and can be considered stealth as well.

Adding a filetype association

To add an association, make sure eXpresso is running, and double click on a file. If eXpresso does not yet have an association for that filetype, it will then display a window asking which app you wish to associate it with. For apps in the PortableApps Format, the filetype should be associated with the launcher, not the base app.

Configuration

Right-clicking the eXpresso system tray icon will open a menu. From here, you can access basic instructions, a copy of the license, and graphical interfaces to add and modify all of eXpresso's settings.

eXpresso Hotkeys

These combinations of keystrokes are shortcuts for different eXpresso functions.
  • Shift + Ctrl + Left Click = Associate Main App with extension
  • Alt + Ctrl + Left Click = Associate Alternative App with extension
  • Ctrl + Alt + c = Pause eXpresso
  • Alt + Win + m = Configure Mouse settings
  • Alt + Win + x = Configure Extension settings
  • Alt + Win + a = Configure App settings
  • Alt + Win + w = Configure Window settings
  • Alt + Win + r = Open eXpresso Run Dialogue
  • Shift + Win + Left Click = Add window to monitored window list
  • Escape + c = exit eXpresso
